# Adel-EUAI Website

Official website for **Adel-EUAI**, showcasing projects and web applications.

---

## Overview

This website is a fully static HTML/CSS site built to showcase:

- Personal projects and web applications
- Privacy Policy and Terms of Service
- General information about Adel-EUAI initiatives

No user accounts, tracking, or analytics are included. This website is designed to be privacy-focused and minimal.

---

## Structure

```text
adel-euai-website/
├── index.html          # Main Landing Page (SEO Optimized)
├── .gitignore          # Git Filter: Excludes sensitive & heavy files
├── privacy.html        # Privacy Policy
├── terms.html          # Terms of Service
├── about.html          # About Adel
├── styles.css          # Main Stylesheets
├── legal.css           # Dedicated Styles for Legal & Compliance Pages
├── legal.html          # Dedicated Styles for Legal & Compliance Pages
├── LICENSE             # Software Licensing Agreement
├── package.json        # Project Metadata
├── README.md           # Project Documentation & Developer Guide
└── asset/              # Branding, Logos & Previews
```
---

## Usage

Open `index.html` in your browser to view the website.  
No build step or server is required as this is a static HTML/CSS website.

```bash
# Open index.html directly in your browser
open index.html
```
